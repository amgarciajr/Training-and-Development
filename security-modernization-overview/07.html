Field Guide — Section 3.3

Immediate Escalation Triggers

Escalate to the CISO the same day you find these — not next week, not in the report.
01
Permanent Global Admin
2+ is a risk. 5+ is a crisis. Permanent GA assignments must be eliminated and replaced with PIM-activated roles.
02
No Emergency Access Accounts
No break-glass accounts exist, or accounts not tested in 90+ days — leaving zero recovery path during lockout.
03
Legacy Auth Actively in Use
SMTP Auth, IMAP, POP3, Basic Auth still active — these protocols bypass MFA entirely and are primary attack vectors.
04
CA Policy — No Legacy Auth Block
Conditional Access policy set exists but no rule blocking legacy auth — the single most common gap found in the field.
05
MFA Registration Below 60%
Any user population with MFA coverage under 60% represents an unacceptable credential compromise surface.
06
Labels Optional, No Enforcement
Sensitivity labels published but entirely optional — no enforcement means no protection. Labels without policy are theater.
07
No DLP in Enforcement Mode
DLP policies exist only in simulation mode — data exfiltration is unblocked. Simulation is not protection.
08
No IR Playbook for Credentials
No incident response playbook exists for credential compromise — the most likely breach scenario has no documented response.
09
Guest Accounts — No Expiration
Guest identities with no expiration policy accumulate indefinitely — unmanaged external access is a persistent exposure.
10
Service Principals — Owner: None
Service principals with no assigned owner are unmanaged, unreviewed, and often over-privileged — a silent attack surface.
