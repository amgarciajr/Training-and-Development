Domain Maturity Scores — Summary

Five-Domain Assessment Baseline  ·  Security Modernization Field Guide §3.4

Maturity Scale Reference
1
Initial No formal process — reactive, active exploitation risk
2
Developing Some controls exist but inconsistently enforced
3
Defined Core controls in place, documented, no active exploit risk
4
Managed Monitored, measured, repeatable, metrics available
5
Optimized Automated, continuous improvement, threat-informed
Framing: These scores represent your starting baseline — not a grade. The question is "Where are we starting from?" not "How did you do?"
